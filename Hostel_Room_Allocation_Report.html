Introduction

The Hostel Room Allocation System is a Java-based desktop application designed to streamline and automate the process of assigning hostel rooms to students in an educational institution. Managing hostel accommodations manually is a tedious, error-prone, and time-consuming task that often leads to misallocation, duplicate entries, and administrative inefficiencies. This project aims to eliminate these issues by providing a computerized solution.

The application provides a user-friendly graphical interface built using Java Swing, backed by an Oracle Database for reliable and persistent data storage. The system enables administrators to register new users, authenticate via a secure login mechanism, allocate rooms to students by specifying their details and preferred room type (Single, Double, or Triple), and view all current allocations in a clean tabular format.

The system follows a modular architecture where each functional screen — Login, Registration, Dashboard, Room Allocation, and View Allocations — is encapsulated in its own Java class. This promotes code organization, maintainability, and ease of future enhancements. A dedicated database connection utility class (DBConnection) centralizes all database access, ensuring consistent and reliable connection handling throughout the application.

The front-end GUI is developed using Java Swing components such as JFrame, JTextField, JPasswordField, JButton, JComboBox, and JTable, laid out with GridBagLayout and BoxLayout. The back-end data layer utilizes Oracle Database accessed through the JDBC API, with PreparedStatement for secure and parameterized SQL queries, preventing SQL injection vulnerabilities.

The primary objective of this project is to provide a simple yet effective tool that allows hostel administrators to manage room allocations efficiently. The system stores student details and room assignments in the database, allowing the data to be queried and displayed at any time. This ensures transparency, accuracy, and accountability in the hostel management process.
